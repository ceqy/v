## 简介

My Admin 是一个基于 Vue 3、Vite、TypeScript 开发的现代化中后台管理系统模板。它采用了最新的前端技术栈，提供了丰富的组件和功能，开箱即用，可快速搭建企业级中后台应用。

## 特性

- **最新技术栈**：基于 Vue 3、Vite、TypeScript 开发
- **TypeScript**：全面支持 TypeScript，类型安全
- **主题定制**：支持多套主题，可自定义主题色彩
- **国际化**：内置完善的国际化方案，支持多语言切换
- **权限管理**：动态路由权限控制，细粒度权限管理
- **组件丰富**：提供丰富的 UI 组件和业务组件
- **响应式设计**：适配不同屏幕尺寸
- **开发体验优秀**：热更新、代码提示、错误检查等


## 安装使用

1. 获取项目代码

```bash
git clone https://github.com/ceqy/v.git
```

2. 安装依赖

```bash
cd v
npm install
```

3. 运行

```bash
npm run dev
```

4. 打包

```bash
npm run build
```

## 如何贡献

非常欢迎你的加入！提一个 Issue 或者提交一个 Pull Request。

**Pull Request 流程：**

1. Fork 代码
2. 创建自己的分支：`git checkout -b feature/xxxx`
3. 提交你的修改：`git commit -am 'feat(function): add xxxxx'`
4. 推送您的分支：`git push origin feature/xxxx`
5. 提交 `pull request`

## Git 贡献提交规范

- `feat` 增加新功能
- `fix` 修复问题/BUG
- `style` 代码风格相关无影响运行结果的
- `perf` 优化/性能提升
- `refactor` 重构
- `revert` 撤销修改
- `test` 测试相关
- `docs` 文档/注释
- `chore` 依赖更新/脚手架配置修改等
- `ci` 持续集成
- `types` 类型定义文件更改
